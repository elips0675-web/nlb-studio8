# Инструкция по установке и настройке WordPress темы

Этот документ поможет вам установить и настроить сгенерированную WordPress тему "NLB Theme".

## 1. Копирование темы на сервер

1.  Найдите на вашем компьютере папку с темой `nlb-wp-theme`.
2.  Подключитесь к вашему веб-серверу, где установлен WordPress (например, через FTP-клиент, такой как FileZilla, или через панель управления хостингом).
3.  Перейдите в директорию `wp-content/themes/` в корневой папке вашего сайта.
4.  Загрузите (скопируйте) всю папку `nlb-wp-theme` с вашего компьютера в эту директорию на сервере.

## 2. Активация темы

1.  Войдите в административную панель вашего WordPress сайта (обычно по адресу `ваш-сайт.com/wp-admin`).
2.  В боковом меню перейдите в раздел **Внешний вид > Темы**.
3.  Вы увидите тему "NLB Theme" в списке доступных тем. Наведите на нее курсор и нажмите кнопку **Активировать**.

## 3. Настройка меню навигации

После активации темы ваше меню может не отображаться. Его нужно создать и назначить.

1.  В административной панели перейдите в раздел **Внешний вид > Меню**.
2.  Если у вас еще нет меню, создайте новое, дав ему имя (например, "Главное меню"), и нажмите **Создать меню**.
3.  Слева выберите страницы, которые вы хотите добавить в навигацию, и нажмите **Добавить в меню**.
4.  Перетащите пункты меню, чтобы расположить их в нужном порядке.
5.  В разделе **Настройки меню** внизу страницы, поставьте галочку в области отображения **Primary**.
6.  Нажмите **Сохранить меню**.

После выполнения этих шагов ваша кастомная WordPress тема будет установлена, активирована и готова к работе, а навигационное меню будет отображаться корректно.
