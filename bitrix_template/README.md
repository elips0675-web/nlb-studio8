# Шаблон для "1С-Битрикс: Управление сайтом"

Этот каталог содержит стартовый шаблон для "1С-Битрикс" на основе Bootstrap 4.

## Важное замечание

Этот шаблон **не является** автоматической конвертацией оригинального Next.js проекта. Проект на Next.js (React) и "1С-Битрикс" (PHP) — это принципиально разные, несовместимые технологии.

Этот шаблон был создан **с нуля** как отправная точка, визуально повторяющая структуру оригинального сайта. Он включает в себя:
- Базовую структуру файлов (`header.php`, `footer.php`).
- Подключение Bootstrap 4.
- Пример вызова меню Битрикс.

## Как использовать

1.  Скопируйте папку `nlb_bootstrap4` в директорию `/local/templates/` вашего сайта на Битрикс.
2.  Зайдите в административную панель Битрикс.
3.  Перейдите в `Настройки` → `Настройки продукта` → `Сайты` → `Список сайтов`.
4.  Откройте настройки вашего сайта и в поле "Шаблон сайта" выберите `NLB Bootstrap 4`.
5.  Сохраните изменения.

Дальнейшая работа (создание контента, стилизация компонентов) должна производиться стандартными средствами "1С-Битрикс".
