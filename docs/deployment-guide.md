
# Руководство: Развертывание проекта на хостинг с cPanel

**Важное замечание:** Этот проект является современным JavaScript-приложением, созданным на React/Next.js. Его нельзя развернуть на хостинге простым копированием файлов, как в случае с обычными HTML-сайтами или PHP-скриптами. Вам необходимо сначала **собрать проект**, чтобы получить оптимизированные статические файлы (HTML, CSS, JS), которые и будут загружаться на хостинг.

Большинство стандартных (и самых дешевых) тарифов cPanel-хостинга предназначены для PHP и могут не поддерживать запуск Node.js-приложений напрямую. Описанный ниже метод — это самый универсальный способ, который сработает на любом веб-хостинге.

---

## Метод: Сборка статического сайта и загрузка через cPanel

Этот метод состоит из двух основных этапов: сборка проекта на вашем локальном компьютере и загрузка результатов на хостинг.

### Шаг 1: Сборка проекта на вашем компьютере

1.  **Убедитесь, что у вас установлен Node.js и npm.** Откройте терминал (командную строку) и выполните команды:
    ```bash
    node -v
    npm -v
    ```
    Если вы видите номера версий, всё в порядке. Если нет, скачайте и установите Node.js с официального сайта [nodejs.org](https://nodejs.org/).

2.  **Установите зависимости проекта.** В терминале перейдите в папку с вашим проектом и выполните команду:
    ```bash
    npm install
    ```
    Эта команда скачает все необходимые библиотеки, которые использует проект.

3.  **Запустите сборку.** После установки зависимостей выполните команду:
    ```bash
    npm run build
    ```
    Эта команда запустит процесс сборки. React/Next.js возьмет все `.tsx` и `.css` файлы, оптимизирует их и создаст готовую для использования папку с названием `out` или `build` в корне вашего проекта. Именно содержимое этой папки вам и нужно будет загрузить на хостинг.

### Шаг 2: Загрузка файлов на хостинг через cPanel

1.  **Заархивируйте результат сборки.**
    - Зайдите в папку, которая появилась после Шага 1 (скорее всего, она называется `out`).
    - Выделите **всё содержимое** этой папки (но не саму папку `out`).
    - Создайте из этих файлов ZIP-архив (например, `project.zip`).

2.  **Войдите в cPanel.** Используйте логин и пароль, предоставленные вашим хостинг-провайдером.

3.  **Откройте "Менеджер файлов" (File Manager).**

    ![File Manager](https://i.imgur.com/g1f2sV4.png)

4.  **Перейдите в корневую папку вашего сайта.** Обычно это `public_html` или папка с названием вашего домена.

5.  **Нажмите кнопку "Отправить" (Upload) в верхней панели.**

    ![Upload Button](https://i.imgur.com/kS9gP9C.png)

6.  **Загрузите ваш ZIP-архив.** Выберите созданный ранее `project.zip` и дождитесь окончания загрузки.

7.  **Вернитесь в Менеджер файлов.** Вы увидите свой загруженный архив.

8.  **Распакуйте архив.** Нажмите на файл `project.zip` правой кнопкой мыши и выберите "Extract" или "Извлечь". Подтвердите извлечение в текущую папку (`public_html`).

    ![Extract Files](https://i.imgur.com/5wQJVeN.png)

9.  **Проверьте результат.** Откройте ваш сайт в браузере. Он должен отображаться корректно.

### Что делать при обновлениях?

Каждый раз, когда вы вносите изменения в код проекта, вам нужно будет **повторить весь процесс заново**: запустить `npm run build` на своем компьютере, заархивировать новое содержимое папки `out` и перезалить его на хостинг, заменив старые файлы.
